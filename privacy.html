Política de Privacidade

Última atualização: Julho de 2026

Esta Política de Privacidade descreve como o aplicativo "Minuto de Paz Automação" lida com informações no contexto da publicação automatizada de conteúdo na conta @minutodepaz_br no TikTok.

1. Dados coletados

O aplicativo utiliza exclusivamente o token de acesso OAuth fornecido pelo TikTok para autorizar a publicação de vídeos na conta @minutodepaz_br. Nenhum dado pessoal de terceiros, seguidores ou espectadores é coletado, armazenado ou processado por este aplicativo.

2. Uso das informações

As únicas informações utilizadas são as necessárias para autenticar e publicar conteúdo de vídeo diretamente na conta autorizada, por meio da Content Posting API do TikTok.

3. Armazenamento e segurança

Tokens de acesso são armazenados de forma segura e criptografada em infraestrutura própria, e não são compartilhados com terceiros.

4. Compartilhamento de dados

Este aplicativo não compartilha, vende ou transfere quaisquer dados a terceiros. Nenhuma informação é utilizada para fins publicitários.

5. Direitos do usuário

O administrador da conta pode revogar o acesso do aplicativo a qualquer momento através das configurações de aplicativos conectados no TikTok.

6. Contato

Para dúvidas sobre esta política, entre em contato através do perfil @minutodepaz_br no TikTok.
